eCommerce Website

React + PHP GraphQL eCommerce app for Scandiweb.

Setup

Backend:
cd NewBackend2/fullstack-test-starter
composer install

Update database config in src/Database/Database.php
Import database
Start web server

Frontend:
cd frontend
npm install
npm run dev

Update API URL in src/services/fetch.ts

Features
Product catalog
Shopping cart
Order placement

Requirements
PHP 8.0+
MySQL
Node.js 18+
